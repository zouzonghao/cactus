### 启用内存缓存

#### 查看 mmagent 配置
 win11
```
mmagent
```
win10
```
get-mmagent
```

#### 开启缓存
```
enable-mmagent -ApplicationPreLaunch
```

#### 关闭内存压缩
```
disable-mmagent -MemoryCompression
```

### 设置缓存最大大小
```
set-mmagent
```
```
8192
```
