## 1、安卓安装 [termux:API](https://github.com/termux/termux-api)

## 2、termux 执行
```sh
pkg install termux-api
```

## 3、起别名
```sh
nano ./zshrc
```
```
alias cc='termux-clipboard-set'  # cc 为输入到剪切板
alias vv='termux-clipboard-get'  # vv 为获取剪切板

# 直接执行剪切板内容
vve() {
    eval "$(vv)"
}
```

## 4、使用

如复制123.cpp文件中的内容到剪贴板即 `cat 123.cpp | cc`

输入 `vv` ，会直接打印剪切板内容

输入 `vve` ，会直接执行剪切板命令
