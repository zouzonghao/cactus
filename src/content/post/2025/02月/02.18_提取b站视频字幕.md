### 1. 打开视频，打开F12开发者工具，网络
### 2. 停用缓存
### 3.1 如果是自动生成的字幕，搜索
  ```
  ai_subtitle
  ```
### 3.2 如果是上传的字幕，搜索
```
json
```
### 4. 选中搜索结果，响应（response）选项卡
### 5. 复制结果，到[字幕转换工具](https://www.dreamlyn.cn/html/bsrt.html)
